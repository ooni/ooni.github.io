Community

Join our global community!

  • Participate in community discussions on Slack or IRC (#ooni@irc.oftc.net).

Tools

Measure internet censorship in your country.

  • Run OONI Probe
    • Mobile App (guide)
    • Desktop App (guide)
    • Command Line Interface (guide)
    • Browser Tool
  • Become an OONI Probe beta tester (Android, iOS).
  • Use OONI Run to coordinate website censorship testing (user guide and documentation on coordinating campaigns).
  • Contribute websites for censorship testing through our Test Lists Editor (guide) or by opening a pull request on GitHub.
  • Translate the OONI Probe apps (guidelines), OONI Run, and OONI Explorer (guidelines).

Data

Investigate internet censorship around the world based on OONI data.

  • Track censorship events worldwide with OONI Explorer (guide).
  • Create charts based on OONI data with the Measurement Aggregation Toolkit (MAT) (guide).
  • Analyze OONI data to investigate internet censorship (tutorial and documentation on interpreting OONI data). Please use the OONI API for light-weight queries, and our Amazon S3 bucket for batch analysis.
  • Join the #ooni-data-analysis channel on Slack or IRC to discuss your findings with other researchers.
